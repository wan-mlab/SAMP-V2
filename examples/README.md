# Examples

Place small example input files here.

Expected CSV format:

```text
sample_id,feature_1,feature_2,feature_3,labels
sample_001,0.13,0.44,0.09,1
sample_002,0.05,0.21,0.31,0
```

The first column is used as the sample index. The `labels` column is required.

The included `example_train.csv` and `example_test.csv` files are small
synthetic files for testing the command-line workflow. They are not benchmark
datasets.
